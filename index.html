Helpdesk Cahul

A small PHP helpdesk project (served with XAMPP / PHP + MySQL).

About

This repository contains a PHP/MySQL helpdesk app. The app must be run on a PHP-enabled web server (for example XAMPP/Apache + MySQL) — GitHub Pages cannot run PHP.

Run locally (recommended)

  1. Install XAMPP (or any PHP+MySQL stack).
  2. Copy this project folder into XAMPP's htdocs (e.g. C:\xampp\htdocs\helpdesk).
  3. Start Apache and MySQL from XAMPP control panel.
  4. Create the database and import any SQL if required (see db.php for DB name).
  5. Open http://localhost/helpdesk/index.php.

Hosting on GitHub

GitHub Pages can host only static files (HTML/CSS/JS). Because this project uses PHP and a database, it cannot be fully hosted on GitHub Pages. Options:

  • Host source on GitHub (recommended) and deploy to a PHP-capable host (DigitalOcean, Heroku with PHP buildpack, shared hosting, etc.).
  • Create a static demo page on GitHub Pages that links to instructions or screenshots.
